# Changelog

## [0.1.0] - 2026-03-18

### Added

- `Cron.parse()` for parsing standard 5-field cron expressions

- `nextAfter()`, `nextN()`, `previousBefore()` time calculations

- `matches()` to check if an instant matches an expression

- `describe()` for human-readable descriptions

- `Cron.isValid()` for expression validation
